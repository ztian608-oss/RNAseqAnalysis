# RNAseqAnalysis Report

## Study Context

- Objective
- Disease or phenotype context
- Focus pathways and markers
- Emphasis mode

## Input Summary

- Counts source
- Sample count and group count
- Design formula
- Contrasts analyzed

## Quality and Overview

- PCA summary
- Low-expression filtering summary, including the `min_count_mean` threshold and number of genes removed
- Notes on replication and model assumptions

## Differential Expression

- Variant selected
- DEG counts per comparison, described relative to the reference group
- Key genes
- Effect-size and significance interpretation

## Trend Clustering

- Whether TCseq was run
- Cluster count
- Major trajectory patterns
- Cluster-specific marker notes

## Functional Enrichment

- Up-regulated gene enrichment summary by comparison
- Down-regulated gene enrichment summary by comparison
- Cluster enrichment summary
- Focus-pathway interpretation

## Stability Assessment

- Variants compared
- Stability metrics
- Why the final variant was selected

## Limitations and Next Steps

- Design or annotation limitations
- Suggested validation analyses
