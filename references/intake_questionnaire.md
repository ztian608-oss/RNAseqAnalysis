# Intake Questionnaire

Collect these answers before running analysis when the user has not already specified them.

## Required

- What is the main research objective?
- Which sample groups should be compared first?
- Is there a preferred reference or control group?
- Do you care more about strict statistical confidence or broader biological interpretation?

## Biological Focus

- Is there a disease, phenotype, treatment, pathway, or cell program of interest?
- Are there marker genes that should be tracked explicitly?
- Are there pathways that should be highlighted even if they are only borderline significant?

## Threshold Preferences

- Preferred `padj` cutoff.
- Preferred `|log2FC|` cutoff.
- Whether to compare multiple threshold settings.

## Trend Analysis

- Are there more than 3 groups or ordered conditions?
- Should temporal or progressive expression patterns be clustered?
- If yes, is interpretability preferred over cluster granularity?

## Reporting

- Which comparisons need the most explanation?
- Should the report prioritize concise conclusions or exhaustive result tables?
